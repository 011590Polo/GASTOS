help Centro de Ayuda

add_circle ¿Cómo agregar un gasto?

  1. Haz clic en el botón "+" flotante o en "Agregar Gasto"
  2. Completa la información requerida (NIC, Nombre, Deuda, Fecha)
  3. Opcionalmente agrega un enlace de pago
  4. Haz clic en "Guardar"

payment ¿Cómo registrar un pago?

  1. Encuentra el gasto en la tabla
  2. Haz clic en "Registrar Pago"
  3. Completa la información del pago
  4. Guarda el registro

filter_list ¿Cómo usar los filtros?

  • Búsqueda: Escribe en la barra de búsqueda para encontrar gastos específicos
  • Estado: Selecciona un estado para filtrar por "Puede Esperar", "Urgente" o "Pagado"

Administrador de Gastos

Versión 2.0

Una aplicación web moderna para gestionar tus deudas y pagos de manera organizada y eficiente.

Características principales:

  • ✅ Gestión completa de gastos
  • 💳 Seguimiento de pagos
  • 🔍 Filtros inteligentes
  • 📊 Estadísticas detalladas
  • 💾 Importar/Exportar datos
  • 📱 Diseño responsive
  • 🎨 Material Design

Tecnologías utilizadas:

  • HTML5, CSS3, JavaScript ES6+
  • Material Design
  • Chart.js para gráficos
  • LocalStorage para datos
